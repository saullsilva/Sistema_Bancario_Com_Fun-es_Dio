# Sistema_Bancario_Com_Fun-es_Dio
## Desafio Dio - Suzano - Python Developer
Implementação de um simulador bancário em Python com menu interativo para operações de depósito, saque (com limites diários) e consulta de extrato. O sistema inclui funcionalidades de cadastro de usuários e criação de contas bancárias vinculadas. O código destaca-se pelo uso de conceitos mais avançados da linguagem nas funções, como a aplicação de argumentos estritamente posicionais (positional-only) e estritamente nomeados (keyword-only).
